== Description ==

Easy to use & customize shortcode for highlighting text in any post, page, or sidebar widget!

== Installation ==

1. Unzip contents of “rad-text-highlighter.zip”.
2. Upload folder “rad-text-highlighter” to your “/wp-content/plugins” directory.
3. Activate plugin through WordPress administration at Dashboard > Plugins.
4. Enjoy!
